# AI Test Case Generator

## Overview
This project is a prototype tool that automatically generates software test cases based on a given feature description.

## Features
- Generates Positive, Negative, and Edge test cases
- Simple CLI-based interface
- Structured JSON output

## Tech Stack
- Python

## How It Works
The system takes a feature description as input and generates different categories of test cases using predefined logic. This can be extended using AI models like OpenAI or LangChain.

## Future Improvements
- Integrate OpenAI API for intelligent test generation
- Build a web interface using Flask
- Add CI/CD integration for automated testing
